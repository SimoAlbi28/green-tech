DOCUMENTO TECNICO
==================

Progetto: Portale GreenTech
Modalità: Caso C (ho usato il template fornito e l'ho integrato in React)


HO USATO

- React per fare il sito (è quello che abbiamo studiato a lezione)
- Bootstrap per lo stile dei bottoni e del form (preso dal template)
- Spring Boot per il backend (sono partito dal semilavorato fornito)
- H2 come database per salvare le proposte 
  (è un database leggero che si avvia insieme al server, 
  senza dover installare programmi extra. 
  Ho scelto questo perchè è più semplice da configurare)


FRONTEND

Ho creato una pagina con queste sezioni:
- Hero verde con titolo e bottone "Partecipa"
- Missione e Visione
- Tecnologie Verdi
- Testimonianze
- Premi
- Form di registrazione con tutti i campi richiesti
- Contatore delle proposte inviate
- Footer

Il form manda i dati al backend quando si clicca "Invia".
Se va tutto bene esce un messaggio di conferma e il form si svuota.
I dati inviati si vedono nella console del browser e in quella del server.


BACKEND

Sono partito dal semilavorato e ho fatto queste modifiche:

- Ho attivato le dipendenze per il database nel pom.xml
- Ho aggiunto H2 come database (salva i dati in un file locale)
- Ho aggiunto le annotazioni alla classe Submission per collegarla al database
- Ho creato il Repository per poter salvare e leggere i dati
- Ho completato il Service che prima non faceva nulla
- Ho completato il Controller per salvare davvero i dati nel database
- Ho aggiunto la stampa dei dati nella console del server

Quando arriva una proposta il server la stampa nella console e la salva nel database.


COME FAR FUNZIONARE IL PROGETTO

1. Avviare il backend:
   cd backend
   mvn spring-boot:run

2. Aprire il browser su http://localhost:8080

3. Compilare il form e cliccare Invia

4. Controllare che esce il messaggio di conferma

5. Controllare nella console del server che i dati vengono stampati


COME VEDERE I DATI NEL DATABASE

1. Aprire http://localhost:8080/h2
2. Mettere come JDBC URL: jdbc:h2:file:./greentech_data
3. User: sa
4. Password: lasciare vuoto
5. Cliccare Connect
6. Scrivere: SELECT * FROM SUBMISSIONS;
7. Cliccare Run per vedere i dati salvati
